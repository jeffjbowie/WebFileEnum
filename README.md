# WebFileEnum

```
usage: WebFileEnum.py [-h] -u "https://example.com/" -w "~/Wordlist.txt" [-e "login,sign-in"] [--v]

Enumerate files & directories with the ability to exclude results based on string-matching against HTML content body.

optional arguments:

-h, --help  show this help message and exit
-u "https://example.com/"
-w "~/Wordlist.txt"
-e "login,sign-in"  Comma-separated list of strings to exclude.
--v Verbose output.
```
